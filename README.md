## Run Locally

Clone the project

```bash
  git clone https://github.com/jeronasiedu/Unifeed-Blog.git
```

Go to the project directory

```bash
  cd Unifeed-Blog
```

Start the server

```bash
  use vscode live server extension to preview the project
```
